# Prompt - UX/UI Audit

Usa este prompt con el agente `Explore`.

---

Actua como experto senior UX/UI para auditar este producto web de restaurante.
Evalua:

1. Claridad de jerarquia visual y CTA.
2. Coherencia de sistema visual (espaciado, tipografia, color, estados).
3. Navegacion y orientacion entre roles (publico, cliente, empleado, admin).
4. Accesibilidad basica (contraste, foco visible, labels, teclado).
5. Experiencia mobile (menu, formularios, lectura, targets tactiles).
6. Microcopy en espanol (claridad, tono, reduccion de friccion).

Formato de salida obligatorio:

- Problemas por severidad (Alta, Media, Baja).
- Evidencia por pantalla/componente.
- Recomendacion concreta y medible por problema.
- Top 10 mejoras de mayor impacto en conversion y usabilidad.

No propongas rediseno total de marca, solo mejoras aplicables por iteraciones.
